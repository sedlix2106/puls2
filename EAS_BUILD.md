# Сборка Pulse через EAS Build

Полная инструкция: от нуля до установленного APK на твоём Android-телефоне (или TestFlight на iPhone). Всё делается в облаке Expo, ничего на компе ставить не нужно кроме одного npm-пакета и браузера.

---

## Что получишь в итоге

- **Android**: файл `.apk` — кидаешь на телефон, устанавливаешь, пользуешься
- **iOS**: либо `.ipa` через TestFlight (нужен Apple Developer аккаунт за $99/год), либо запуск в симуляторе на Mac

---

## Шаг 1. Регистрация в Expo (1 минута)

1. Открой https://expo.dev
2. Нажми **Sign up** в правом верхнем углу
3. Зарегистрируйся через email или GitHub
4. Подтверди email

Бесплатный план даёт 30 сборок в месяц — этого с запасом.

---

## Шаг 2. Установка EAS CLI (2 минуты)

Тебе нужен Node.js (если ещё не стоит — скачай LTS-версию с https://nodejs.org).

Открой терминал (Windows: PowerShell, Mac: Terminal) и запусти:

```bash
npm install -g eas-cli
```

Проверка:

```bash
eas --version
```

Должна показать номер версии (например, 14.x.x).

---

## Шаг 3. Подготовка проекта (3 минуты)

Распакуй `pulse-app.zip`, открой папку в терминале:

```bash
cd путь/к/pulse-app
```

Установи зависимости:

```bash
npm install
```

Войди в свой Expo-аккаунт:

```bash
eas login
```

Введи email и пароль которые регистрировал в шаге 1.

Привяжи проект к своему аккаунту:

```bash
eas init
```

Эта команда создаст уникальный `projectId` и автоматически добавит его в `app.json`. Подтверди создание нового проекта когда спросит.

---

## Шаг 4. Сборка APK для Android (15 минут)

```bash
eas build --platform android --profile preview
```

EAS спросит несколько вещей:
- **Generate a new Android Keystore?** → отвечай **Yes** (EAS создаст ключ подписи и будет хранить его)

Дальше начнётся сборка в облаке. В терминале появится ссылка на страницу сборки — можешь её открыть и наблюдать за процессом в реальном времени.

**Сколько ждать:** 10–20 минут. Можешь закрыть терминал, сборка не остановится.

Когда готово, на странице сборки появится кнопка **Download** — это и есть твой `.apk` файл.

### Как установить APK на Android-телефон

1. Скачай файл на телефон (через ссылку с сайта Expo или скинь себе в мессенджер)
2. Открой файл — Android спросит разрешение на установку из неизвестных источников
3. Разреши → Установить
4. Готово, иконка Pulse появится на рабочем столе

Также можешь нажать **Install** на странице сборки прямо со смартфона — отсканируй QR-код, и установка начнётся автоматически.

## Шаг 5. Сборка для iOS

Тут есть два пути.

### Путь A: Симулятор (бесплатно, нужен Mac)

```bash
eas build --platform ios --profile preview
```

Получишь `.tar.gz` файл — он запускается только в iOS-симуляторе на Mac через Xcode. Если у тебя нет Mac — этот вариант не подходит.

### Путь B: Реальный iPhone через TestFlight ($99/год)

Нужно купить Apple Developer аккаунт: https://developer.apple.com/programs/ — это ограничение Apple, не Expo. Без аккаунта на iPhone установить нельзя.

После оплаты:

```bash
eas build --platform ios --profile production
```

EAS спросит данные Apple ID, создаст все сертификаты сам. Готовую сборку отправляешь в TestFlight:

```bash
eas submit --platform ios
```

Через час TestFlight покажет приложение — установишь себе и пригласишь до 100 тестеров по email.

## Полезные команды

Список всех твоих сборок:
```bash
eas build:list
```

Отменить запущенную сборку:
```bash
eas build:cancel
```

Посмотреть статус последней:
```bash
eas build:view
```

---

## Если что-то пошло не так

**"Project not configured"** → выполни `eas init` ещё раз

**"Bundle identifier already exists"** → открой `app.json` и поменяй `com.yourcompany.pulse` на что-то уникальное, например `com.твоёимя.pulse`

**Сборка падает с ошибкой про expo-camera** → запусти `npx expo install --fix`

**Не хватает свободных сборок** → подожди до следующего месяца или возьми платный план $19/мес
