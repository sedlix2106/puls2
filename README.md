# Pulse — мессенджер на React Native

Приложение с реальным чатом и звонками между устройствами.

## Что внутри

- ✅ **Регистрация и вход** — Firebase Authentication по email/паролю
- ✅ **Чат** — Firestore с live-синхронизацией между устройствами
- ✅ **Голосовые звонки** — LiveKit + WebRTC, реальный голос
- ✅ **Видеозвонки** — LiveKit + WebRTC, реальное видео
- ✅ **Входящий звонок** — экран с вибрацией и принять/отклонить
- ✅ **Профиль** — с настоящими данными пользователя
- ✅ **Контакты** — список зарегистрированных пользователей для нового чата

## Порядок настройки

Настраивается за **30 минут** при первом подключении. Дальше всё работает.

### 1. Firebase (для чата)

См. **`FIREBASE_SETUP.md`** — пошаговая инструкция.

Что нужно: создать проект, включить Authentication и Firestore, скопировать 6 ключей в `src/firebase/config.js`.

### 2. LiveKit (для звонков)

См. **`CALLS_SETUP.md`** — пошаговая инструкция.

Что нужно: зарегистрироваться на LiveKit Cloud, создать ключ, подставить в `src/livekit/config.js`.

### 3. Сборка приложения

См. **`EAS_BUILD.md`** — пошаговая инструкция.

Что нужно: для звонков обязательно делать **development build** через EAS, не обычный Expo Go.

```bash
npm install
eas login
eas build --platform android --profile development
```

Через 15 минут получишь APK с поддержкой WebRTC. Установи на Android-телефон.

Запуск разработки:

```bash
npx expo start --dev-client
```

## Как тестировать звонки

Нужно **два устройства**:

1. На обоих установлен development build
2. Зарегистрируй два разных аккаунта
3. На первом: «+» → выбери второго → шапка чата → 📞 или 📹
4. На втором: появится экран входящего → принять
5. Говори, видь, слышь

## Что осталось до production

- **Push-уведомления** для звонков и сообщений когда приложение закрыто
- **Сервер** для генерации токенов LiveKit (сейчас Secret в клиенте)
- **Картинки и файлы** в чате через Firebase Storage
- **Иконка приложения** и сплеш-скрин (сейчас плейсхолдеры)
- **Публикация** в Google Play и App Store

Любое из этого можно добавить отдельно.

## Бесплатные лимиты

- **Firebase**: 50 000 пользователей, миллионы операций — бесплатно
- **LiveKit Cloud**: 50 000 минут разговоров в месяц — бесплатно

Для пет-проекта или маленькой команды денег не понадобится.
